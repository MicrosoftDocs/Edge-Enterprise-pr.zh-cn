## <a name="external-threat-protection"></a><span data-ttu-id="b8aea-111">外部威胁防护</span><span class="sxs-lookup"><span data-stu-id="b8aea-111">External threat protection</span></span>

### <a name="highest-rated-protection-against-phishing-and-malware"></a><span data-ttu-id="b8aea-112">抵御网站钓鱼和恶意软件的最高级别防护</span><span class="sxs-lookup"><span data-stu-id="b8aea-112">Highest-rated protection against phishing and malware</span></span>

<span data-ttu-id="b8aea-113">根据 NSS Labs 的一项独立研究，相比 Google Chrome 的安全浏览，内置在 Microsoft Edge 中的 SmartScreen 可阻止更多的[网络钓鱼](https://query.prod.cms.rt.microsoft.com/cms/api/am/binary/RWASN1)和[恶意软件](https://query.prod.cms.rt.microsoft.com/cms/api/am/binary/RWANMW)企图。</span><span class="sxs-lookup"><span data-stu-id="b8aea-113">Built into Microsoft Edge, SmartScreen blocks more [phishing](https://query.prod.cms.rt.microsoft.com/cms/api/am/binary/RWASN1) and [malware](https://query.prod.cms.rt.microsoft.com/cms/api/am/binary/RWANMW) attempts than Google Chrome's Safe Browsing, according to an independent study from NSS Labs.</span></span> <span data-ttu-id="b8aea-114">SmartScreen 可在用户联机工作时提供网站和下载的实时信誉检查，它是 [Microsoft Intelligent Security Graph](https://www.microsoft.com/microsoft-365/windows/intelligent-security) 的一部分，它汲取从 Microsoft 庞大的全球资产、研究人员和合作伙伴网络产生的信号和见解。</span><span class="sxs-lookup"><span data-stu-id="b8aea-114">SmartScreen provides real-time reputation checks of sites and downloads as users work online, and is part of the [Microsoft Intelligent Security Graph](https://www.microsoft.com/microsoft-365/windows/intelligent-security), which draws signals and insights generated from Microsoft's large network of global assets, researchers, and partners.</span></span> <span data-ttu-id="b8aea-115">通过对基于云的动态危险网站和下载列表进行检查，Microsoft Edge 有助于检测和阻止快速消失的短暂威胁。</span><span class="sxs-lookup"><span data-stu-id="b8aea-115">By running checks against dynamic, cloud-based lists of dangerous sites and downloads, Microsoft Edge helps to detect and block even ephemeral threats that quickly disappear.</span></span>
